# 사각 (Sa-gak) — 나만의 감정 아카이브로 책 읽기 동반자

## 서비스 개요

사각은 종이책 독자가 읽는 순간의 감정을 기록·축적해 나만의 아카이브로 키우고, 같은 책을 읽는 사람들과 느슨하게 연결되는 독서 동반자 앱으로, 커뮤니티가 0명이어도 혼자 읽는 사람에게 가치가 있도록 "표현·아카이브"를 우선시하며 연결은 그 위에 얹는다. 바쁜 현대인들이 책 읽기 시작하는 장벽을 낮추고, 독서의 과정에서 생기는 감정과 생각을 저장하며, 다른 독서자와의 공감을 통해 독서를 지속하는 경험을 제공한다.

## 타겟 사용자

> **참고**: 본 타겟 정의는 기획 문서 기반이며, `brand/target-audience.md` 구체화 후 해당 문서가 단일 진실 원천이 됩니다.

종이책 독서를 하고 싶지만 바쁜 일상으로 시작이 어렵거나 중도 포기가 잦은 사람들, 지식/토론 중심 독서모임의 부담(비용, 과제, 전문성 경쟁)을 피하고 싶은 사람들, 긴 독후감 대신 짧은 감정 기록(단어/한 문장)과 공감 위주의 소통을 선호하는 사람들, 혼자 읽되 특정 순간에는 즉시 누군가와 감정을 나누고 싶은 사람들에게 집중된 서비스다. 주요 사용자는 25-40대 직장인으로, 책을 읽고 싶은 의지는 있지만 시간 부족과 지속의 어려움을 겪는 이들이며, 독서를 하나의 취미나 성취감 있는 활동이 아닌 일상의 일부로 만들고 싶어 한다.

## 핵심 기능

- **종이책 서재 관리**: 바코드 스캔/수동 검색으로 책 등록, 진도 추적, 독서 상태 관리

## 진행 상태

**완료된 SPEC**:
- **SPEC-DB-001**: 백엔드 구현 완료 (Supabase 스키마, API 엔드포인트, 인증 시스템)
- **SPEC-UI-001**: 프론트엔드 기반 구현 완료 (Expo SDK 55 + React 19.2 + RN 0.83.2 + TypeScript strict, 디자인 토큰, ThemeProvider + useTheme + useManualMode, 6가지 컴포넌트, Jest 테스트 72/72, 커버리지 93.68%, 웹 업로드 검증)

**남은 작업 (향후 SPEC에서 구현)**:
- 실제 화면 구현 (홈/서재/클럽/마이 화면)
- 네비게이션 라우팅 연결
- Supabase 클라이언트 통합
- 인증 플로우 구현
- **감정 아카이브**: 페이지별 감정 기록(단어/한 줄) 누적 및 완독 다이어리 자동 생성
- **Track A 합류형**: 읽고 있는 사람에게 "같이 읽어요" 요청 보내기
- **Track B 개설형**: 0명이어도 출발하는 함께 읽기 모임 생성
- **스포일러 방지 진도별 피드**: 현재 진도 기반으로 블러 처리된 콘텐츠 피드
- **안전한 글쓰기 울타리**: 단어 질문지, 공감 스티커 반응을 통한 쉬운 표현
- **아날로그 루틴 헬퍼**: 다정한 독서 알림, 독서 타이머 등 독서 습관 지원
- **감정 아카이브 기반 보상**: 기록 누적을 통한 성취감 제공과 재미 요소

## 주요 사용 시나리오

1. **독서 지원**: 첫 페이지를 펼치는 시작의 주저함을 다정한 알림과 첫 기록 보상으로 극복. 사용자는 책을 펼치고 앱을 열면 "오늘의 첫 페이지가 당신을 기다리고 있어요"라는 메시지와 함께 첫 감정 기록을 할 때 보상을 받아 시작의 마법을 경험한다.

2. **감정 공유**: 서툰 감상을 스티커로 공감받으며 표현 욕구 충족. 특정 페이지에서 "서글픔"이나 "감동" 같은 단어를 기록하면 다른 독서자의 공감 스티커가 도착하고, "나도 같은 페이지에서 뭔가 느꼈어요"라는 연결감을 얻는다.

3. **비동기 연결**: 실시간 매칭이 아닌 요청 기반(Track A)으로 같은 책 독자를 찾아 느슨하게 연결. 특정 페이지의 감정 기록(emotion_records)을 공개하면 같은 구간을 읽는 독자가 자연스럽게 공감·합류 요청으로 연결된다.

4. **완독 성취**: 페이지별 아카이브가 완독 다이어리로 자동 발행되며 성취감 제공. 책을 마칠 때까지 모든 감정 기록이 모여 "이 책과의 여정"이라는 다이어리로 완성되고, 사용자는 자신의 독서 여정을 시각적으로 확인하며 성취감을 얻는다.

## 가치 제안

사각은 기존 독서 모임의 딱딱한 토론, 강제 독후감, 높은 비용 장벽을 타파하고 지식 경쟁이 아닌 감정 공유와 다정한 동행으로 독서 경험을 지속시킨다. "나는 책을 읽는 사람"이라는 정체성을 실제 독서 결과물로 증명하는 가치를 제공하며, 독서를 의무나 경쟁이 아닌 자기 발견과 감정의 여정으로 만든다. 특히 사회적 부담 없이 혼자서도 책을 읽을 수 있으며, 필요할 때만 연결되는 유연한 구조로 독서의 즐거움과 편안함을 동시에 제공한다.

## 수익화 전략

현재 단계에서는 수익화보다는 데이터 축적을 우선으로 한다. 사용자 완독률, 감정 반응 패턴, 독서 습관 데이터를 축적하여 미래의 다양한 수익 모델을 준비한다. 잠재적 수익화 경로로는 프리미엄 기능 유료화(고급 분석, 맞춤 추천), 출판사 B2B 데이터 제공(독자 반응 분석), 동네 서점 연계(로컬 북클럽), 독서 챌린지 스폰서십 등을 고려 중이며, 모든 수익 모델은 사용자 경험 저해 없이 자연스럽게 통합될 것을 전제로 한다.
